API类目：mb.workorder.batch.get

API描述：批量工单查询

版本号：1.0

系统级别输入参数

参数名称 参数值 参数描述
appKey *
access_token *

应用级别输入参数

startDate 查询三个月内工单创建时间开始，格式：yyyy-MM-dd HH:mm:ss
endDate 查询三个月内工单创建时间结束，格式：yyyy-MM-dd HH:mm:ss
workorderState 工单状态（0：录入中 1：处理中 2：处理完成 3：处理异常）
workorderType 工单类型 0->缺货
pageNo 页码。取值范围：大于零的整数；默认值：1
pageSize 每页条数。取值范围：大于零的整数；默认值：30；最大值:100
sortType 排序方式。默认按升序，1：按降序
