名称 类型 示例值 是否必须 描述

content  String  1234 是 工单描述
reply_content  String  1234 是 回复内存
exception_reason  String  1234 是 异常原因，如：订单已取消、订单已发货、ERP异常等
remark  String  1234 是 工单备注
order_id  String  1234 是 订单ID
mb_order_id  String  1234 是 美邦订单ID
vendor_id  String  1234 是 商家ID
oos_items  List<OosItem>  1234 是 缺货项
last_modified_user  String  ${field.defaultValue} 是 ${field.desc}
