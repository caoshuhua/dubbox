API列表>工单>批量工单查询

API功能描述

API接口：mb.workorder.batch.get

功能：批量工单查询

授权类型：需要

应用场景：通用应用 自研应用

系统级别输入参数

名称 类型 是否必须 描述
appkey String 是 应用的app_key
token String 是 签名
timestamp String 是 时间戳，格式为yyyy-MM-dd HH:mm:ss，例如：2011-06-16 13:23:30。美邦API服务端允许客户端请求时间误差为6分钟

应用级别输入参数

名称 类型 是否必须 示例值 描述
start_date  String  否 2016-02-01 17:15:00   查询三个月内工单创建时间开始，格式：yyyy-MM-dd HH:mm:ss 
end_date  String  否 2016-02-15 17:15:00   查询三个月内工单创建时间结束，格式：yyyy-MM-dd HH:mm:ss 
workorder_state  String  否 0   工单状态（0：录入中 1：处理中 2：处理完成 3：处理异常） 
workorder_type  String  否 0   工单类型 0->缺货 
page_no  String  否 1   页码。取值范围：大于零的整数；默认值：1 
page_size  String  否 100   每页条数。取值范围：大于零的整数；默认值：30；最大值:100 
sort_type  String  否 1   排序方式。默认按升序，1：按降序 

返回结果

名称 类型 示例值 描述
resp_code String 0 0为正常，其它值参见错误代码一览表
resp_info String 成功   成功 或者 失败信息  
total_results  int  1234   工单信息总数 
work_orders  List<WorkOrder>  1234   工单信息列表
