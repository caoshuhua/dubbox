沙箱环境操作手册

一、沙箱环境介绍

唯品会沙箱环境是唯品会开放平台提供给独立软件开发商（ISV）的测试环境。沙箱环境模拟线上真实环境，ISV可以在沙箱中仿真的测试VOP接口的功能。

二、沙箱环境操作说明

一，获取沙箱的使用权限

请到支持中心进行沙箱的APPKEY和SECRET的申请。申请流程：提交问题--》技术咨询--》沙箱测试。

进入支持中心

二，沙箱环境入口

接口访问地址：sandbox.vipapis.com

三，接口调用方式

沙箱环境的开发规范和生产环境一致，请参考以下文档：
开发规范

四，沙箱环境API权限包

开发者在沙箱环境可以调用的API权限包是保持和生产环境一致的权限范围。如果开发者在权限方面有特殊需要，可以直接联系我们的供应商支持人员。我们会提供相应的帮助。
