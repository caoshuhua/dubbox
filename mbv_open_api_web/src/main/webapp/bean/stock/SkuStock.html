对象名：SkuStock

数据字段说明

名称 类型 示例值 是否必须 描述
sku_id  String  111111 是 商品编码
vendor_sku_id  String  12345678912 否 商家货号（SKU）
vendor_international_id  String  11111111111 否 国际码
available_stock  String  2 否 可用库存
actual_stock  String  2 是 实际库存,注意实际库存不能小于当前已分配库存
committed_stock  String  2 否 已分配库存
locked_stock  String  2 否 已锁定库存
