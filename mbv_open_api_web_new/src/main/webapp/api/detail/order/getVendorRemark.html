API列表>订单>查询商家备注

API功能描述

API接口：mb.order.vendorremark.get

功能：查询商家备注

授权类型：需要

应用场景：通用应用 自研应用

系统级别输入参数

名称 类型 是否必须 描述
appkey String 是 应用的app_key
token String 是 签名
timestamp String 是 时间戳，格式为yyyy-MM-dd HH:mm:ss，例如：2011-06-16 13:23:30。美邦API服务端允许客户端请求时间误差为6分钟

应用级别输入参数

名称 类型 是否必须 示例值 描述
order_id  String  是 000032280   订单ID 

返回结果

名称 类型 示例值 描述
resp_code String 0 0为正常，其它值参见错误代码一览表
resp_info String 成功   成功 或者 失败信息  
vendor_remark  String  1234   商家备注
