简要描述

对象名：OrderInfo

数据字段说明

名称 类型 示例值 是否必须 描述
id  Long  1234 是 内部使用ID
order_id  String  1234 是 订单ID
order_create_date  String  1234 是 订单生成时间
order_state  String  1234 是 订单状态（1：待发货 2：已发货 8：已撤销 9：失效 16：已挂起）
vendor_id  String  1234 是 商家ID
warehouse_id  String  1234 是 发货仓库ID
total_quantity  int  1234 是 总出库量
total_value  double  1234 是 总金额
consignee  String  1234 是 收货人
moblie  String  1234 是 收货人手机号码
country  String  1234 是 国家
province  String  1234 是 省份
city  String  1234 是 城市
district  String  1234 是 地区
deliv_address  String  1234 是 送货地址
deliv_pstd  String  1234 是 邮政编码
tracking_number  String  1234 是 快递单号
express_company_id  String  1234 是 快递公司
mb_order_id  String  1234 是 美邦订单ID
vendor_remark  String  1234 是 商家备注
order_remark  String  1234 是 订单备注
channel_code  String  1234 是 渠道编码
channel_name  String  1234 是 渠道名称
pay_time  String  1234 是 订单付款时间
is_oos  String  1234 是 是否已发起缺货工单 1：是；2：否
items  List<ItemInfo>  1234 是 订单项
